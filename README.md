# work_with_stagesepx

about how to use stagesepx in production

该项目旨在介绍如何将 [stagesepx](https://github.com/williamfzc/stagesepx) 落地到生产项目中发挥作用。所有的例子均开箱即用，可以在应用中逐渐增进理解，避免止步在前期准备阶段。
